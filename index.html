git-pivotal by trydionel

A collection of git utilities to ease integration with Pivotal Tracker.

Requirements

Git Flow.

Download

Download in either zip or tar formats, or clone the project with Git by running: 

$ git clone git://github.com/iortega/git-pivotal

Configuration

Global 

$ git config --global pivotal.api-token 9a9a9a9a9a9a9a9a9a9a
$ git config --global pivotal.full-name "Ignacio Ortega"

Git project 

$ git config -f .git/config pivotal.project-id PROJECT_ID

Git usage

Getting a story: 

$ git feature

Getting an specific story: 

$ git feature --story-id=STORY_ID

Finish a story: 

$ git finish
